# 07 — Copy Voice & Anti-Patterns

The site's voice is half the design. Get the language wrong and a perfectly built section feels off. This is the calibration reference.

## The voice in one sentence

German, declarative, photographic, professional but unembellished — Matthias is a curator stating what he does, not a vendor pitching a service.

## Voice characteristics

- **German, formal-but-warm.** Mostly uses *Du*/*Sie*-neutral phrasing (no direct address most of the time — statements, not pitches). Where direct address appears, it's *Du* in the contact lead-in ("Schreibe kurz…") and on conceptual pages.
- **Declarative.** Verb-led sentences. "Ich arbeite mit…", "Wir klären…", "Bilder werden geplant, nicht improvisiert." Not "Sie können bei mir…", "Wir bieten Ihnen…".
- **Photographic vocabulary.** *Linse, Komposition, Licht, Lack, Chrom, Patina, Silhouette, Mechanik, Rahmen, Bühne, Bildbühne, Archiv, Kontaktbogen, Bildraum, Kuratierung, Editorial, Stage, Lens, Aperture.* Domain words signal expertise.
- **Quality vocabulary.** *Kuratiert · sauber · ruhig · präzise · technisch sauber · einheitlich · bereit für Print · ohne Effektpose · ohne glattbügeln · nicht beliebig.* These adjectives recur. They're the brand's quiet promise.
- **Anti-superlative.** No "amazing", "best", "stunning", "world-class", "premium". Never. The work speaks; the copy under-claims.
- **Anti-corporate.** No "Lösungen", "Innovationen", "Mehrwert", "synergetisch", "ganzheitlich". No buzzwords. No "passionate about".
- **Anti-marketing.** No "Jetzt unverbindlich anfragen!", no "Sichern Sie sich…", no "schon ab €X". The contact CTA is *"Anfrage senden"* or *"Projekt anfragen"*. Period.
- **Short sentences, sometimes elliptical.** "Location · Location · Detail." is a complete utterance. "Sechs Bereiche. Eine Linse." is a complete H1. Fragments are fine and frequent in kickers and one-line statements.
- **Numbered hierarchy.** Lists use `01`, `02`, `03` (zero-padded, mono kicker style) — not bullets. Implies sequence, intentionality, archive.

## Signature lines (use as voice anchors)

From real pages:

- "Bilder, die einen Raum verändern."
- "Sechs Bereiche. Eine Linse."
- "Geschwindigkeit und Leidenschaft auf Bildern."
- "Speed ohne Klischee."
- "Nähe ohne Beliebigkeit."
- "Der Oldtimer als Exponat."
- "Maschine mit Haltung."
- "Landschaft als Bildraum."
- "Ein Motorrad muss auch im Stand Spannung tragen."
- "Keine generischen Fahrzeugbilder: kontrollierte Reflexe, ruhige Linien und ein Bildsatz, der technisch sauber weiterverwendet werden kann."
- "Lack, Chrom, Leder und Patina werden nicht nostalgisch überhöht, sondern mit Charakter präzise dokumentiert."
- "Fotografie aus Düsseldorf — kuratiert für Marke, Sammlung und Druck. Editorial geführt, technisch ruhig, bereit für die nächste Ausgabe."
- "Beschreibe kurz, welcher Bereich passt — Automobil, Sportwagen, Oldtimer, Motorrad, Portrait oder Landschaft — und wofür die Bilder genutzt werden sollen."
- "Nicht aufgelistet? Trotzdem schreiben."
- "Location · Location · Detail · Web/Print"

These aren't suggestions; they're the calibration. New copy that sounds like the same person wrote it is right. New copy that sounds like a different writer is wrong.

## Headline formulas

- **"X. Y."** — two short statements, with a period between, no conjunction. "Sechs Bereiche. Eine Linse." "Speed. Klar gezeichnet." Often paired with the italic-serif `<em>` on one word.
- **"X als Y."** — re-framing as. "Der Oldtimer als Exponat." "Landschaft als Bildraum." "Maschine als Bühne."
- **"X ohne Y."** — defining by negative. "Nähe ohne Beliebigkeit." "Speed ohne Klischee."
- **"X auf Bildern."** — concretizing an abstract noun. "Geschwindigkeit und Leidenschaft auf Bildern."
- **"X mit Y."** — pairing two qualities. "Maschine mit Haltung."

## Lead-paragraph formulas

- **City-anchored lede:** "{Topic} {City}: {service pitch}. {city-specific geography clause}, {what that enables}, {without what failure}."
  - Example: "Motorrad Fotografie in Köln: Schärfe auf Maschine und Haltung — Silhouette, Mechanik, Details und Fahrerbilder werden als kraftvolle Serie geplant, nicht als beliebiges Bike-Foto. Rhein, Innenstadt, Rheinauhafen, Messeumfeld und urbane Hinterhöfe liefern starke Kontraste, ohne die Serie in Kulissen-Kitsch zu ziehen."

- **Topic-fixed lede (no city):** "{Topic abstract claim}. {Concrete what-we-do}. {Negative — what it isn't}."
  - Example: "Portraits als ruhige visuelle Haltung — Blick, Distanz und Licht so geführt, dass ein Bild professionell wirkt, ohne den Menschen dahinter glattzubügeln."

- **Service / module lede:** "{Action verb}. {Concrete inputs}. {Concrete outputs}."
  - Example: "Fahrzeuge, Reflexe, Linien und Details als präzise Bildserie für Verkauf, Marke, Sammlung oder Kampagne."

## Contact-form lead-in formula

```
"Schreibe kurz, {what we need to know about the project} {city-specific clause if applicable} — und wir klären {what comes next} {without what worry}."
```

Examples:
- Motorrad/Köln: "Schreibe kurz, welches Bike in Köln fotografiert werden soll — Modell, Standort, ob Fahrerbilder gewünscht sind und wofür die Bilder genutzt werden. Wir klären Licht, Location und Ablauf gemeinsam."
- Portrait/Essen: "Schreibe kurz, wofür die Portraits in Essen gedacht sind — Personal Branding, Editorial, Team, Presse oder Bewerbung. Wichtig sind Person/Team, gewünschte Wirkung, Ort und Zeitraum. Stil und Licht klären wir vor dem ersten Klick."

## CTA labels (memorize)

- Primary CTA (anywhere): **"Anfrage senden →"** or **"Projekt anfragen →"** or topic-specific: **"Motorrad-Shooting anfragen →"**, **"Oldtimer-Shooting anfragen →"**, **"Portrait-Shooting anfragen →"**, **"Sportwagen-Serie anfragen →"**, **"Landschaftsserie anfragen →"**, **"Automobil-Shooting anfragen →"**.
- Secondary CTA: **"Portfolio ansehen"**, **"Portfolio"**, **"Serien öffnen →"**.
- Navigation CTA (in topbar): **"Projekt anfragen"** (no arrow).
- Service-row arrow: **→** (a literal Unicode arrow, not "More info" or "Details").

Never use: "Get a quote", "Book now", "Start", "Learn more", "Discover", "Explore", "Find out".

## Anti-patterns — what NOT to write

### Forbidden phrases

These are jarring against the site's voice:

- "Bei uns finden Sie…"
- "Wir bieten Ihnen…"
- "Lassen Sie sich begeistern!"
- "Hochwertige Lösungen für…"
- "Unsere Leidenschaft ist…"
- "Mit über XX Jahren Erfahrung…"
- "Jetzt unverbindlich anfragen!"
- "Sichern Sie sich einen Termin!"
- "Top-Qualität zu fairen Preisen"
- "Exklusive Angebote"
- "Premium-Service"
- "Innovative Bildkonzepte"
- Any superlative without a measurable anchor

### Forbidden punctuation moves

- **No exclamation marks** anywhere except the legal pages. Statements don't shout.
- **No question marks in headlines** (FAQ heading questions are fine — actual interrogatives).
- **No emojis.** Not in copy, not in nav, not in alt text.
- **No "®" or "™"** symbols.

### Forbidden section types

(Already in `02-page-architecture.md`, restating for the writer-brain.)

- No testimonial sliders with star ratings.
- No "trusted by" / "as seen in" logo strip. (A "Kunden & Auftraggeber" editorial colophon is OK if framed as credit, not as social proof.)
- No newsletter signup boxes.
- No popups, exit-intent, modals.
- No live chat widgets.
- No price tables ("Starter / Pro / Enterprise").
- No "Why choose us" bulleted list.
- No timer-based scarcity.
- No "100% Zufriedenheitsgarantie" badges.

### NOT forbidden — clarifying common confusion

These look like cards at first glance but are actually editorial patterns the site uses extensively. Don't reject them on sight:

- **3-up or 4-up image grids** with a label underneath (mono micro-eyebrow + bold sans label + tag-line). The "Sechs Bereiche" pattern. Allowed and central.
- **4-up image gallery** on light/paper with consistent gap. "Ausgewählte Arbeiten" pattern. Allowed.
- **Contact-sheet strip** (4-col uniform 6px gap). Allowed.
- **Archive wall** (column-count masonry, near-zero gap). Allowed.

The line between "editorial grid" (allowed) and "card grid" (forbidden) is **chrome**, not geometry:
- Sharp corners (`border-radius: 0`) → allowed.
- No drop shadows → allowed.
- Hover effect is `filter` or `opacity` only, never `transform: scale()` → allowed.
- Hover doesn't change colors playfully (e.g., red overlay on hover) → allowed.

Add card chrome (radius, shadow, scale-bounce, gradient) → forbidden, drift away from the site's voice.

## When the user asks for copy

1. Pick the closest existing page and steal its skeleton.
2. Replace only the topic-/city-/role-specific words.
3. Use the formulas in this file for headlines and ledes.
4. Run the result through the anti-patterns list — every red flag is a fix.
5. Read it aloud. If it sounds like a vendor pitch, rewrite as a curator's statement.

## When the user asks for English copy

The site is currently German-only. If the user wants English (international clients, agency outreach), the voice translates as:

- "Curated", "considered", "precise", "quiet", "clear" — same anti-flair register.
- "Photography from Düsseldorf, calibrated for brand, collection, and print."
- "Six fields. One lens."
- "Cars as a visual system."
- Active voice, present tense, short sentences.

But discuss with the user before adding English — bilingual pages add maintenance overhead and the SEO cluster doesn't extend to English yet.

## Quick voice test

Ask of any copy you've written:

1. Could a luxury brand consultancy have written this? → If yes, *too generic*. Rewrite with photographic specifics.
2. Does it use "wir bieten" / "we offer"? → If yes, rewrite as a declarative statement.
3. Does it include an exclamation mark or superlative? → If yes, remove.
4. Does it have at least one photographic noun (Linse, Lack, Kontaktbogen, etc.) or quality adjective (kuratiert, sauber, ruhig, präzise)? → If no, add one.
5. Could the same paragraph appear on any competitor's site? → If yes, replace with something specific to this practice.
